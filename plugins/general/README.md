# jebs-dev-tools:general

General development utilities for Claude Code.

## Features

### Skills (1)
- **mermaid-diagrams** - Creating software diagrams with Mermaid syntax

## Installation

```bash
cc --plugin-dir /path/to/plugins/general
```

## Usage

The mermaid-diagrams skill triggers when you need to:
- Create class diagrams for domain modeling
- Create sequence diagrams for application flows
- Create flowcharts for processes
- Create ERD diagrams for database schemas
- Create C4 architecture diagrams
